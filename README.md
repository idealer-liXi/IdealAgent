# IdealAgent

IdealAgent is a learning-oriented full-stack recreation of MiniAgent.

It keeps MiniAgent's main architecture and feature shape:

- Spring Boot 3 + Java 17 backend.
- DDD-style Maven modules.
- MyBatis + MySQL for business data.
- PostgreSQL pgvector for RAG embeddings.
- Redis, RabbitMQ, Docker, and OneAPI for local learning infrastructure.
- Vue 3 + Vite + TailwindCSS frontend.
- MCP server examples for Amap, Bocha, CSDN, Email, and WeCom.

## Learning Milestones

3. AI API/model/client/prompt/advisor/MCP configuration.
4. Chat Client end-to-end flow.
5. RAG file and Git import.
6. MCP tool integration and MCP servers.
7. Work Agent strategy execution.
8. Workspace pages and APIs.
9. Admin panel pages and APIs.
10. Scheduled tasks, cache, statistics, MQ, and deployment polish.

## Current Status

Milestone 1 builds the foundation only. Business APIs are added in later milestones.
