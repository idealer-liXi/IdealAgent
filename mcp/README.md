# IdealAgent MCP Servers

This directory recreates MiniAgent's MCP learning services.

Planned services:

- `mcp-server-amap`: weather lookup.
- `mcp-server-bocha`: web search.
- `mcp-server-csdn`: CSDN article publishing.
- `mcp-server-email`: email sending.
- `mcp-server-wecom`: WeCom message sending.

Implementation is added in the MCP milestone after Chat and RAG are working.
